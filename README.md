# 💌 Date Invite Site

A little website to ask someone on a date. The "No" button runs away. 😄

## How to put it online (free)

### Easiest — GitHub Pages
1. Go to github.com and sign in (make a free account if needed).
2. Click **+** (top right) → **New repository**. Name it `date`, set **Public**, Create.
3. Click **Add file → Upload files**. Drag in **index.html**. Commit.
4. Go to **Settings → Pages**. Source: branch **main**, folder **/(root)**. Save.
5. Wait ~1 min. Your link appears: `https://YOURNAME.github.io/date/`
6. Send that link. 💕

### Fastest — Netlify Drop (no account, 30 seconds)
1. Go to app.netlify.com/drop
2. Drag **index.html** onto the page.
3. Copy the live link it gives you. Done.

## Editing
Open `index.html` in any text editor. Easy things to change near the top:
- The title and headlines (add her name!)
- Colors: the `--rose` and `--rose-deep` values
- Food choices: the `data-food` lines

Good luck! 🥹
